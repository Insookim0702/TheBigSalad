자바스크립트 배열 비교/삭제/대입 방법

by devebucks 2020. 12. 19. 수정 삭제
728x90

안녕하세요.

회사에서 vue를 사용하고 있고, front-end 위주의 개발을 하고 있습니다.

회사에서 과제를 진행하면서 자바스크립트에 대해서 더 깊이 있게 공부하고 있습니다.

 

과정에서, api를 통해서 넘겨받은 데이터를 자바스크립트 단에서 미리 자료구조를 처리하는 과정이 필요해서 자연스럽게 배열을 많이 사용하게 되었고, 실무에서 사용할 만한 개념은 명확히 알고 가고자, 정리하는 글을 쓰게 되었습니다.

 

첫 번째로 알아볼 것은  두 개의 배열의 동등 비교입니다.

배열에 원소가 모두 같은 지를 확인하는 방법은 다음과 같습니다.

- for문을 사용해서 원소들을 일일히 비교하는 방법.

- join()을 사용해서 배열의 모든 원소를 문자열로 만들어서 그 문자열을 비교연산자를 사용해서 비교하는 방법.

 

저는 주로,  join()메서드를 사용해서 비교를 합니다. 코드의 길이도 짧고 for문을 돌리는 것보다 더 가벼운 퍼포먼스를 보일 것 같기 때문입니다.

코드는 다음처럼 사용합니다.

let resultOptionValueArray = []
for (let q = 0; q <= i; q++) {
   resultOptionValueArray.push(this.allOptions[k].attributes[q].value)
}
let resultOptionValueString = resultOptionValueArray.join('/')

 

두 번째로 알아볼 것은 splice()메서드를 사용한 배열을 원하는 인덱스에 추가/삭제/대입하는 방법입니다.

기본적으로 splice()메서드을 알아보겠습니다. MDN를 참고했습니다.

Array.prototype.splice()입니다.

splice()메서드는 배열의 기존 요소를 삭제 또는 교체하거나 새 요소를 추가하여 배열의 내용을 변경합니다.

배열.splice(start, deleteCount )

start(필수)
배열의 변경을 시작할 인덱스 : 배열의 길이보다 큰 값이 입력되면 -> 배열의 길이로 설정됨.
음수라면 -> 배열의 끝에서 부터 요소를 세어나갑니다.

deleteCount(옵션)
배열에서 제거할 요소의 수.
생략가능.
deleteCount > 배열크기 -  start보다  -> 모든 요소 제거
deleteCount = 0 이하일 경우 -> 어떤 요소도 제거되지 않음.

item1, item2, ... (옵션)
배열에 추가할 요소임.
아무것도 추가하지 않으면 splice()는 요소를 제거하기만 함.

 

1. 추가

배열.splice(추가할 요소가 들어갈 인덱스, 0, 배열에 넣을 값)

ex)

var mybook = ['오브젝트', '코딩테스트다', '스프링']
mybook.splice(1, 0, '반응형 웹 만들기')
//result
//mybook = ['오브젝트', '반응형 웹 만들기', '코딩테스트다', '스프링']

 

2. 삭제

배열.splice(삭제할 인덱스, 삭제할 요소 개수)

var mybook = ['오브젝트', '반응형 웹 만들기', '코딩테스트다', '스프링']
mybook.splice(0, 1)
//result
//mybook = ['반응형 웹 만들기', '코딩테스트다', '스프링']


var mybook = ['오브젝트', '반응형 웹 만들기', '코딩테스트다', '스프링']
mybook.splice(0, 3)
//result
//mybook = ['스프링']

 

3. 대입

배열.splice(대입할 요소 인덱스, 대입할 요소 인덱스부터 삭제할 요소 개수, 대입할 요소)

var mybook = ['오브젝트', '코딩테스트다', '스프링']
mybook.splice(1, 1, 'sssssssss')
//result
//mybook = ['오브젝트', 'sssssssss', '스프링']
